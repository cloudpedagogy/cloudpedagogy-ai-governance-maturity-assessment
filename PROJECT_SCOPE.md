# Project Scope: AI Governance Maturity Assessment

## Goal
Build a fully functional static local-first web application for assessing AI governance maturity.

## Core Functions
1. **Assessment Framework**: 7 dimensions (Traceability, Boundary Clarity, Risk Management, Accountability, Policy Alignment, Review Practices, Learning Capability).
2. **Maturity Scoring**: 0-4 scales with descriptive levels.
3. **Maturity Levels**: Emerging → Developing → Established → Advanced.
4. **Gap Identification**: Detect weak areas and imbalances.
5. **Pattern Interpretation**: Analyze high-level governance patterns.
6. **Improvement Pathways**: Suggest focus areas for improvement.
7. **Reflection Layer**: Embed reflective prompts throughout the assessment.
8. **Maturity Profile Output**: Generate structured summaries and exports.

## Constraints
- Static site (HTML/CSS/JS).
- No backend/database.
- No external APIs.
- Privacy-preserving.
